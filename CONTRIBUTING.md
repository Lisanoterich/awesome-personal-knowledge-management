# Contributing

Thanks for helping build the best PKM tools list!

## How to Add a Tool

1. **Check it fits** — Must be a PKM tool (not a general note app, not a task manager)
2. **Research it** — Verify pricing, features, and language support from official sources
3. **Open a PR** — Add the tool to the appropriate category in both README.md and README.zh-CN.md

## Format

Each tool entry must include:
- Name + website link
- Paradigm (one sentence describing the core approach)
- Pricing (verified)
- Languages supported
- 3-5 strengths
- Best for / Not for (honest assessment)

## Quality Standards

- No affiliate links
- No AI-generated descriptions
- Pricing must be verified within last 3 months
- "Free" means genuinely free, not "free trial"
